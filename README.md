# Spin Pong
A fun little Java application to play Pong with cool animations, spin physics, and real-time sphere rendering.

Usage
---
### Dependencies
- JDK version 17+

### Windows
You can use the included runMe.bat

### Build (All Platforms)
Download the project and compile the class files:

```
javac Paddle.java  
javac PlayerBall.java  
javac GamePanel.java  
javac GameFrame.java  
javac Main.java  
java Main  
```

To run the game with specific dimensions, run Main with width and height arguments. By default, Spin Pong is launched at 1280x720.

```
java Main [width] [height]
```

> Spin Pong is only optimized for 1280x720 and 1920x1080
